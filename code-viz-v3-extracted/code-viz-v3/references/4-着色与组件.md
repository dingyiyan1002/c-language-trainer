# 语法着色速查 + 更多 CodeLine 示例

## 颜色对照表（背下来）

| Tailwind 类 | 用于 | 示例 |
|---|---|---|
| `text-blue-400` | 关键字/类型 | `int` `char` `void` `return` `if` `else` `while` `for` `struct` `const` `static` `sizeof` `typedef` `enum` `switch` `case` `break` `unsigned` `signed` `long` |
| `text-purple-400` | 预处理器指令 | `#include` `#define` `#ifdef` `#ifndef` `#endif` `#pragma` |
| `text-green-300` | 字符串 / 头文件 | `<stdio.h>` `"hello"` `'A'` `<stdlib.h>` |
| `text-orange-300` | 数字字面量 | `10` `0xFF` `3.14` `0b1010` `-1` `100L` |
| `text-yellow-300` | 函数名 | `printf` `main` `scanf` `malloc` `free` `strlen` 及自定义函数 |
| `text-pink-400` | 运算符 | `=` `+` `-` `*` `&` `->` `==` `!=` `<` `>` `>=` `<=` `&&` `\|\|` `!` `++` `--` `+=` `-=` |
| `text-gray-500 italic` | 注释 | `// 注释` `/* 注释 */` |
| （无类名） | 普通变量名、括号、分号 | `a` `p` `(` `)` `{` `}` `;` |

## 常见代码行着色示例（直接复制）

```jsx
{/* #include <stdio.h> */}
<CodeLine num={1}><span className="text-purple-400">#include</span>{' '}<span className="text-green-300">&lt;stdio.h&gt;</span></CodeLine>

{/* #define MAX 100 */}
<CodeLine num={2}><span className="text-purple-400">#define</span>{' '}MAX{' '}<span className="text-orange-300">100</span></CodeLine>

{/* 空行 */}
<CodeLine num={3}>{' '}</CodeLine>

{/* int main() { */}
<CodeLine num={4}><span className="text-blue-400">int</span>{' '}<span className="text-yellow-300">main</span>() &#123;</CodeLine>

{/* int a = 10; */}
<CodeLine num={5}>&nbsp;&nbsp;<span className="text-blue-400">int</span>{' '}a <span className="text-pink-400">=</span>{' '}<span className="text-orange-300">10</span>;</CodeLine>

{/* int *p = &a; */}
<CodeLine num={6}>&nbsp;&nbsp;<span className="text-blue-400">int</span>{' '}<span className="text-pink-400">*</span>p <span className="text-pink-400">=</span>{' '}<span className="text-pink-400">&amp;</span>a;</CodeLine>

{/* *p = 20; */}
<CodeLine num={7}>&nbsp;&nbsp;<span className="text-pink-400">*</span>p <span className="text-pink-400">=</span>{' '}<span className="text-orange-300">20</span>;</CodeLine>

{/* int b = a + *p; */}
<CodeLine num={8}>&nbsp;&nbsp;<span className="text-blue-400">int</span>{' '}b <span className="text-pink-400">=</span>{' '}a <span className="text-pink-400">+</span>{' '}<span className="text-pink-400">*</span>p;</CodeLine>

{/* printf("%d\n", b); */}
<CodeLine num={9}>&nbsp;&nbsp;<span className="text-yellow-300">printf</span>(<span className="text-green-300">"%d\n"</span>, b);</CodeLine>

{/* scanf("%d", &n); */}
<CodeLine num={9}>&nbsp;&nbsp;<span className="text-yellow-300">scanf</span>(<span className="text-green-300">"%d"</span>, <span className="text-pink-400">&amp;</span>n);</CodeLine>

{/* return 0; */}
<CodeLine num={10}>&nbsp;&nbsp;<span className="text-blue-400">return</span>{' '}<span className="text-orange-300">0</span>;</CodeLine>

{/* } */}
<CodeLine num={11}>&#125;</CodeLine>

{/* if (a > 0) { */}
<CodeLine num={5}>&nbsp;&nbsp;<span className="text-blue-400">if</span> (a{' '}<span className="text-pink-400">&gt;</span>{' '}<span className="text-orange-300">0</span>) &#123;</CodeLine>

{/* } else { */}
<CodeLine num={7}>&nbsp;&nbsp;&#125; <span className="text-blue-400">else</span> &#123;</CodeLine>

{/* while (i < n) { */}
<CodeLine num={6}>&nbsp;&nbsp;<span className="text-blue-400">while</span> (i{' '}<span className="text-pink-400">&lt;</span>{' '}n) &#123;</CodeLine>

{/* for (int i = 0; i < n; i++) { */}
<CodeLine num={5}>&nbsp;&nbsp;<span className="text-blue-400">for</span> (<span className="text-blue-400">int</span>{' '}i <span className="text-pink-400">=</span>{' '}<span className="text-orange-300">0</span>; i{' '}<span className="text-pink-400">&lt;</span>{' '}n; i<span className="text-pink-400">++</span>) &#123;</CodeLine>

{/* struct Node *head = malloc(sizeof(struct Node)); */}
<CodeLine num={8}>&nbsp;&nbsp;<span className="text-blue-400">struct</span> Node{' '}<span className="text-pink-400">*</span>head <span className="text-pink-400">=</span>{' '}<span className="text-yellow-300">malloc</span>(<span className="text-blue-400">sizeof</span>(<span className="text-blue-400">struct</span> Node));</CodeLine>

{/* // 注释 */}
<CodeLine num={3}>&nbsp;&nbsp;<span className="text-gray-500 italic">// 初始化变量</span></CodeLine>

{/* node->next = NULL; */}
<CodeLine num={10}>&nbsp;&nbsp;node<span className="text-pink-400">-&gt;</span>next <span className="text-pink-400">=</span>{' '}NULL;</CodeLine>

{/* arr[i] = arr[j]; */}
<CodeLine num={7}>&nbsp;&nbsp;arr[i] <span className="text-pink-400">=</span>{' '}arr[j];</CodeLine>
```

## explanations 图标速查

可用的 icon 字段值：
`info` `check` `alert` `bolt` `rocket` `lightbulb` `memory` `pointer` `skull` `stack` `lock` `folder` `target` `terminal`

推荐搭配：
- `info` + `blue` — 一般讲解步骤
- `memory` + `yellow` — 变量声明/内存分配
- `pointer` + `purple` — 指针操作
- `skull` + `red` — 危险操作（如越界、悬空指针）
- `bolt` + `orange` — 关键运算
- `check` + `green` — 正常结束/输出
- `terminal` + `green` — printf/输出
- `stack` + `blue` — 函数调用/栈操作
- `lock` + `purple` — 互斥/同步
