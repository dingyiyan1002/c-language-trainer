【编程题】

1、实现字符串的排序。（输入hello world good，输出good hello world，其中字符串个数任意）

2、输入两个有序的字符串（从小到大），合并成一个有序的字符串。（输入cdhxyz fjln 输出cdfhjlnxyz）

3、命令行参数实现字符串排序。

4、一个整型数组里除了两个数字之外，其他的数字都出现了两次。请写程序（异或）找出这两个只出现一次的数字。

5、编写一个函数来查找字符串数组中的最长公共前缀。（输入abca  abc  abca  abc  abcc 输出 abc）

6、给定一个字符串，字符串是有序的整数（位数不限）集合，逗号相连，移除相同的数字，使每个数字只出现一次，输出最终的数字个数。例如输入：0,0,1,1,1,2,2,31,31,444   输出 5

【写出结果】

#include <stdio.h>

int main()
{
	char *c[] = {“ENTNG”, “NST”, “AMAZI”, “FIRBE”};
	char **cp[] = {c + 3, c + 2, c + 1, c};
	char ***cpp = cp;

	printf(“%s”, **++cpp);
	printf(“%s ”, *--*++cpp + 3);
	printf(“%s”, *cpp[-2] + 3);
	printf(“%s”, cpp[-1][-1] + 1);
	printf(“\n”);
	return 0;
}
