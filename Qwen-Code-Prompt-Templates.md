# 🎯 Qwen Code 高级提示词模板集合

***

## 📋 文档说明

这里收集了多个实用的提示词模板，适用于不同的编程场景。

**使用方法：**

1. 根据场景选择合适的模板
2. 复制并修改为你的具体需求
3. 发送给 Qwen Code
4. 如果跳过步骤，提醒"请遵循流程"

***
