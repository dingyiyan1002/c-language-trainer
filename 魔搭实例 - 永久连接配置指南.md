## 🎯 方案二：本地 VSCode Remote SSH（适合深度开发）

### ✅ 优点
- 使用本地 VSCode，习惯的环境
- 可以使用本地插件和配置
- 文件传输方便
- 适合长时间开发

### 📝 配置步骤

#### 第 1 步：获取 SSH 连接命令

在魔搭社区控制台找到你的实例，点击"连接信息"，应该会显示类似：

```bash
ssh -p <端口> root@<跳板机 IP>
# 或
ssh root@<实例域名> -p 22
```

#### 第 2 步：配置本地 SSH

编辑本地 SSH 配置文件 `~/.ssh/config`（Windows: `C:\Users\你的用户名\.ssh\config`）：

```
Host modelscope-dsw
    HostName 1723758.dsw.cn-hangzhou.aliyuncs.com
    User root
    Port 22
    IdentityFile ~/.ssh/id_ed25519
    # 如果需要跳板机，添加：
    # ProxyJump user@bastion-host:port
```

#### 第 3 步：在 VSCode 中连接

1. 安装 **Remote - SSH** 插件
2. 按 `F1` → 输入 `Remote-SSH: Connect to Host`
3. 选择 `modelscope-dsw`
4. 首次连接需要输入密码（从控制台获取）

## 🎯 与 Qwen 对话的快速方法

### 方法 1：直接运行 Python

在终端执行：

```bash
python3 -c "
from modelscope.pipelines import pipeline
chat = pipeline(task='chat', model='Qwen/Qwen-7B-Chat')
print('Qwen: 你好！有什么可以帮你的吗？')
while True:
    text = input('你：')
    if text == 'quit': break
    if text.strip():
        result = chat(text)
        print(f'Qwen: {result[\"response\"]}')
"
```

### 方法 2：创建对话脚本

创建文件 `chat.py`：

```python
#!/usr/bin/env python3
from modelscope.pipelines import pipeline

print("🤖 Qwen 对话机器人（输入 quit 退出）\n")
chat = pipeline(task='chat', model='Qwen/Qwen-7B-Chat')

while True:
    text = input("你：")
    if text == 'quit':
        break
    if text.strip():
        result = chat(text)
        print(f"Qwen: {result['response']}\n")
```

运行：
```bash
python3 chat.py
```

## ❓ 常见问题

### Q1: 网页 VSCode 打不开？
- 检查网络连接
- 清除浏览器缓存
- 尝试无痕模式
- 换用 Chrome/Edge 浏览器

### Q2: 连接超时怎么办？
- 检查实例是否运行中（控制台查看）
- 可能需要重启实例
- 联系魔搭技术支持

### Q3: 文件会丢失吗？
- 实例运行期间文件不会丢失
- 实例释放后数据会清空
- 重要代码请推送到 GitHub

### Q4: 如何保存工作？
- 使用 Git 推送到远程仓库
- 或下载到本地备份
- 或使用魔搭的存储挂载功能
