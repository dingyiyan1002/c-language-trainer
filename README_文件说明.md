# 📦 广深嵌入式岗位分析项目 - 文件说明

## ✅ 已生成文件列表

### 1. 核心分析文件

#### 📄 `广深嵌入式岗位分析与面试指南.md` ⭐ 主文档
**内容**: 完整的面试准备指南，包含：
- 📊 200 个岗位 JD 数据分析

- 📚 10 大面试考点体系（带详细子主题和代码示例）

- 📈 行业趋势分析

**适合人群**: 准备面试的嵌入式开发者

---

### 2. 数据文件

#### 📊 `embedded_jobs_200.csv` - 岗位原始数据
**格式**: CSV（可用 Excel 打开）
**内容**: 200 个嵌入式岗位的详细信息
**字段**:
- company: 公司名称
- location: 工作地点（广州/深圳）
- position: 岗位名称
- salary: 薪资范围
- experience: 经验要求
- education: 学历要求
- skills: 技能要求（分号分隔）
- responsibilities: 岗位职责
- requirements: 任职要求
- post_date: 发布时间（2025-12 或 2026-01）
- high_frequency: 是否高频岗位

**用途**: 
- 查看具体岗位要求
- 数据分析
- 制作简历时参考关键词

---

#### 📈 `skills_analysis.json` - 技能频率分析
**格式**: JSON
**内容**: 所有技能的出现频率统计
**结构**:
```json
{
  "C": {
    "count": 115,
    "percentage": 57.5,
    "importance": "核心"
  },
  "C/C++": {
    "count": 45,
    "percentage": 22.5,
    "importance": "加分"
  },
  ...
}
```

**用途**:
- 了解哪些技能最重要
- 确定学习优先级
- 简历技能关键词优化

---

#### 📝 `interview_topics.json` - 面试主题
**格式**: JSON
**内容**: 10 大面试主题及子主题详细分布
**结构**:
```json
{
  "C 语言核心": {
    "weight": 100,
    "subtopics": [
      {
        "name": "指针与内存",
        "questions": 50,
        "difficulty": "⭐⭐⭐"
      },
      ...
    ]
  },
  ...
}
```

**用途**:
- 了解面试考点分布
- 按权重安排复习时间
- 评估题目难度

---

#### 🗺️ `knowledge_system.json` - 知识体系
**格式**: JSON
**内容**: 完整的体系化知识结构
**包含**:
- metadata: 元数据（版本、地区、时间等）
- learning_path: 5 阶段学习路径
- priority_matrix: 优先级矩阵（must/should/nice to know）
- interview_checklist: 面试检查清单

**用途**:
- 制定长期学习计划
- 检查知识盲区
- 跟踪学习进度

---

### 3. 工具文件

#### 🛠️ `embedded_job_collector.py` - 数据收集器
**功能**: 
- 生成 200 个示例岗位数据
- 分析技能频率
- 生成面试主题
- 构建知识体系
- 导出数据到 CSV/JSON

**运行方式**:
```bash
python embedded_job_collector.py
```

**输出**:
- embedded_jobs_200.csv
- skills_analysis.json
- interview_topics.json
- knowledge_system.json

**扩展**: 可以修改代码连接真实招聘 API 或爬虫

1. **C 语言核心** - 100% 必考

4. **项目经验** - 90% 必考

### 薪资水平（深圳）
- 应届：12-20K
- 1-3 年：15-30K
- 3-5 年：20-45K（华为/大疆/腾讯）
- 5-10 年：30-55K
- 10 年+: 40-70K+

### 学习路径（12-18 个月）
1. **基础阶段**（1-2 月）: C 语言 + 数据结构
2. **进阶阶段**（2-3 月）: 操作系统 + Linux 编程
3. **专业阶段**（3-4 月）: ARM + 驱动开发
4. **实战阶段**（2-3 月）: RTOS + 项目实战
5. **冲刺阶段**（1 月）: 面试准备 + 手撕代码

## 🎯 使用建议

### 准备面试（1-2 个月）
1. 阅读 `广深嵌入式岗位分析与面试指南.md`
2. 重点复习前 5 大考点（C 语言、数据结构、操作系统、项目、手撕代码）
3. 使用 `interview_topics.json` 检查知识盲点
4. 参考 `embedded_jobs_200.csv` 优化简历关键词

### 系统学习（6-18 个月）
1. 参考 `knowledge_system.json` 制定学习计划
2. 按照 5 阶段学习路径逐步推进
3. 定期回顾 `skills_analysis.json` 确保学习方向正确
4. 积累项目经验，对照岗位要求

### 职业规划
1. 查看 `embedded_jobs_200.csv` 了解市场需求
2. 参考薪资分析确定期望
3. 关注趋势分析中的热门方向（车载电子、AIoT 等）
