# 🚀 VSCode Remote SSH 连接魔搭社区完整指南

## 📋 第一步：获取 SSH 密码

1. **打开浏览器访问**：https://modelscope.cn/
2. **登录账号**（如果没有需要先注册）
3. **进入控制台**：
   - 点击右上角头像 → 选择"控制台"
4. **找到你的实例**：
   - 点击左侧菜单"我的实例"
   - 找到你创建的 DSW 实例
5. **获取连接信息**：
   - 点击实例名称或"连接信息"按钮
   - 复制显示的 **SSH 密码**（保存好，后面要用）

---

## 🔧 第二步：安装 VSCode Remote-SSH 插件

1. **打开 VSCode**
2. **打开扩展面板**：
   - 按 `Ctrl + Shift + X`（Windows）
   - 或点击左侧活动栏的扩展图标（四个方块）
3. **搜索插件**：
   - 在搜索框输入：`Remote - SSH`
   - 找到 Microsoft 官方的插件
4. **安装插件**：
   - 点击"安装"按钮
   - 等待安装完成

---

## 🔐 第三步：配置 SSH 连接

### 方法 A：直接连接（推荐新手）

1. **打开命令面板**：
   - 按 `F1` 或 `Ctrl + Shift + P`
2. **输入命令**：
   - 输入：`Remote-SSH: Connect to Host`
   - 选择出现的选项
3. **输入连接信息**：
   - 输入：`root@121.196.175.238`
   - 按回车
4. **选择 SSH 配置文件**：
   - 选择第一个选项（默认 SSH 配置文件）
5. **输入密码**：
   - 在弹出的密码框中输入第一步获取的密码
   - 按回车
6. **确认连接**：
   - 等待连接成功
   - 左下角会显示 `SSH: 121.196.175.238`

### 方法 B：配置 SSH 配置文件（推荐长期使用）

1. **打开命令面板**：
   - 按 `F1` 或 `Ctrl + Shift + P`
2. **打开 SSH 配置**：
   - 输入：`Remote-SSH: Open SSH Configuration File`
   - 选择第一个选项
3. **添加配置**：
   ```
   Host modelscope-dsw
       HostName 121.196.175.238
       User root
       Port 22
   ```
4. **保存文件**：
   - 按 `Ctrl + S` 保存
5. **连接**：
   - 按 `F1` → 输入 `Remote-SSH: Connect to Host`
   - 选择 `modelscope-dsw`
   - 输入密码

---

## ✅ 第四步：验证连接

连接成功后，你可以：

1. **打开终端**：
   - 按 `Ctrl + `` ` （反引号）
   - 或点击菜单栏"终端" → "新建终端"
2. **查看服务器信息**：
   ```bash
   # 查看系统信息
   uname -a
   
   # 查看 Python 版本
   python3 --version
   
   # 查看当前目录
   pwd
   
   # 查看文件列表
   ls -la
   ```

## 🎯 第五步：开始开发

现在你可以：

1. **打开远程文件夹**：
   - 点击"打开文件夹"
   - 选择服务器上的目录（如 `/root`）
2. **创建项目**：
   - 新建文件、编写代码
   - 所有操作都在远程服务器上执行
3. **使用终端**：
   - 直接在 VSCode 终端运行命令
   - 安装依赖、运行程序

## 🔑 可选：配置免密登录（高级）

如果你想跳过每次输入密码的步骤：

### 1. 在本地生成 SSH 密钥（如果还没有）

打开本地 VSCode 终端（不是远程）：
```bash
ssh-keygen -t ed25519 -C "vscode-modelscope"
```
- 一路按回车即可

### 2. 查看公钥内容

```bash
cat ~/.ssh/id_ed25519.pub
```
- 复制显示的内容（以 `ssh-ed25519` 开头）

### 3. 连接到服务器后上传公钥

在远程服务器终端执行：
```bash
# 创建 .ssh 目录
mkdir -p ~/.ssh

# 编辑 authorized_keys 文件
nano ~/.ssh/authorized_keys
```

### 4. 粘贴公钥

- 将第 2 步复制的公钥内容粘贴进去
- 按 `Ctrl + O` 保存
- 按 `Ctrl + X` 退出

### 5. 设置权限

```bash
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
```

### 6. 测试免密登录

断开连接后重新连接，应该不需要密码了！

## ❓ 常见问题

### Q1: 连接超时？
- 检查网络连接
- 确认服务器状态正常
- 尝试重启实例

### Q2: 密码错误？
- 确认从控制台复制的是最新密码
- 注意大小写和特殊字符
- 尝试重置密码

### Q3: 插件无法安装？
- 检查 VSCode 版本（建议最新版）
- 检查网络连接
- 手动下载插件安装

### Q4: 中文乱码？
在服务器终端执行：
```bash
export LANG=zh_CN.UTF-8
export LC_ALL=zh_CN.UTF-8
```
