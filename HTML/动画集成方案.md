**布局结构：**
```
┌──────────────────────────────────────────────┐
│  题目：malloc 动态分配  [🎬 动画演示]       │
├───────────────┬──────────────────────────────┤
│               │                              │
│   代码区      │      动画演示区              │
│   (50%)       │      (50%)                   │
│               │                              │
│   #include    │  [动画未开始时显示预览图]    │
│   int main()  │  或                          │
│   ...         │  [实时动画演示]              │
│               │                              │
├───────────────┴──────────────────────────────┤

```

**动画区状态：**
1. **初始状态** - 显示预览图 + "播放动画"按钮
2. **播放状态** - 实时演示代码执行过程
3. **暂停状态** - 显示暂停界面

## 📋 完整实现流程

### Step 1: 修改题目数据结构

**原结构：**
```typescript
interface Question {
  id: string;
  title: string;
  difficulty: number;
  knowledge: string[];
  code: string;
  type: 'input' | 'choice' | 'blank';
}
```

**新结构：**
```typescript
interface Question {
  id: string;
  title: string;
  difficulty: number;
  knowledge: string[];
  code: string;
  type: 'input' | 'choice' | 'blank';
  animation?: {        // ← 新增动画字段
    enabled: boolean;  // 是否启用动画
    file: string;      // 动画文件路径
    title: string;     // 动画标题
  };
}
```

---

### Step 2: 在题目组件中添加按钮

**React 组件示例：**
```tsx
function QuestionCard({ question }) {
  const [showAnimation, setShowAnimation] = useState(false);
  
  return (
    <div className="question-card">
      <div className="question-header">
        <h2>{question.title}</h2>
        <div className="question-meta">
          <span>难度：{'⭐'.repeat(question.difficulty)}</span>
          <span>知识点：{question.knowledge.join('、')}</span>
        </div>
        
        {question.animation?.enabled && (
          <button 
            className="animation-btn"
            onClick={() => setShowAnimation(true)}
          >
            <LightbulbIcon />
            动画演示
          </button>
        )}
      </div>
      
      <div className="question-body">
        <pre className="code-block">{question.code}</pre>
        
        {question.type === 'input' && (
          <div className="input-area">
            <textarea placeholder="输入程序的输出结果：" />
            <button>提交答案</button>
          </div>
        )}
      </div>
      
      {/* 动画弹窗 */}
      {showAnimation && (
        <AnimationModal 
          file={question.animation.file}
          title={question.animation.title}
          onClose={() => setShowAnimation(false)}
        />
      )}
    </div>
  );
}
```

---

### Step 3: 动画弹窗组件

```tsx
function AnimationModal({ file, title, onClose }) {
  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={e => e.stopPropagation()}>
        <div className="modal-header">
          <h3>🎬 {title}</h3>
          <button className="close-btn" onClick={onClose}>×</button>
        </div>
        <div className="modal-body">
          <iframe 
            src={file} 
            width="100%" 
            height="600px"
            frameBorder="0"
            title={title}
          />
        </div>
      </div>
    </div>
  );
}
```

### 批量添加动画映射

**脚本示例：**
```javascript
// 遍历所有题目，添加动画映射
questions.forEach(question => {
  // 检查是否有对应的动画文件
  const animationFile = `HTML/${question.chapter}/${question.id}.html`;
  
  if (fs.existsSync(animationFile)) {
    question.animation = {
      enabled: true,
      file: animationFile,
      title: `${question.title} - 动画演示`
    };
  }
});
```

## ✅ 总结

### 推荐实现方式

**方案：** 按钮 + 弹窗

**理由：**
- ✅ 不改变现有题目布局
- ✅ 学生可选择是否看动画
- ✅ 动画全屏展示，效果好
- ✅ 实现简单，维护方便

### 实施步骤

1. **修改题目数据结构** - 添加 animation 字段
2. **在题目组件添加按钮** - 判断是否有动画
3. **创建弹窗组件** - 显示动画
4. **添加 CSS 样式** - 美化按钮和弹窗
5. **批量关联动画** - 遍历题目，建立映射
