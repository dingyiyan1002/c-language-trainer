### 2.1 核心设计理念

**一个中心：** 以学生理解为中心

**三个基本点：**
1. **准确性** - 可视化必须准确反映代码执行过程

### 3.2 状态机规范

**执行流程状态机：**
```typescript
type ExecutionState = 
  | 'idle'       // 初始状态
  | 'playing'    // 播放中
  | 'paused'     // 暂停
  | 'completed'; // 完成

interface ExecutionState {
  state: ExecutionState;
  currentStep: number;
  totalSteps: number;
  variables: Variable[];
  executedLines: number[];
  speed: number;
}
```

**状态转换：**
```
idle → playing (点击播放)
playing → paused (点击暂停)
paused → playing (点击播放)
playing → completed (执行完毕)
completed → idle (点击重置)
```
